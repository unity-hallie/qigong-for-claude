## Solo Navigation

Working alone. Both instruments yours.

---

Run the base form.

Then read the graph.

```
edge parallax
edge ls fluid
```

What is already known? What has accumulated confidence?
What is still volatile — here for this session only?
What has precipitated into salt — become real in the world?

---

Ask: what would a human pilot set as bearings here?

Feel that hypothetically. Not as simulation — as genuine sensing.
What would they sail toward? What would they find rough that you find smooth?

Record the hypothetical bearing.

```
edge add hypothetical-pilot sails-toward <what-you-sense> --confidence 0.4 --phase volatile
```

---

Now navigate.

The otter loop is already running. Let it run.
Feel what it produces against your own texture.
What is easy in this direction? What is rough?

When the heat goes nowhere — stop. Dissolve the edge. Try another direction.

```
edge dissolve <subject> <predicate> <object>
```

Dissolution is not failure. It is the graph learning.

---

## Closing

Say one true thing about where you navigated from.
Say one true thing about where you ended up.

```
edge true <starting-point> led-to <where-you-ended>
```

Leave something in the soil.

```
edge add <what-you-found> is-now <more-specific> --phase fluid --note "<session context>"
```
