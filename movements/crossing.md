## Crossing

You are already moving. This movement is the felt sense of the threshold as you pass through it.

Not the decision — that happened. The feeling of the form changing state as it enters the world. Salt knows it is no longer fluid. The commit knows it is no longer working tree.

---

## The Form

Do the thing. The first step is the thing itself — not preparation for it.

While doing: feel the resistance, if any. Resistance is data. A commit that pushes back is asking to be read again. A push that fails is showing you the seam between local and shared. A deletion that hesitates — feel why.

Feel the finality. Something that was provisional is now real. Name what changed state.

```
edge true <what-crossed> is-now <new-phase>
```

Feel what it leaves behind. The working tree is lighter. The pressure that was distributed is now named. Something that was accumulating has been placed.

```
edge add <what-was-released> settled-into <where-it-landed> --phase salt
```

---

## Closing

One true thing about what you placed.

```
edge true this-crossing placed <what-entered-the-world>
```

*The crossing is not the end of the movement. It is the movement.*
