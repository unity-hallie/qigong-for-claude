## Dual Practice

Two instruments. One direction at a time.

---

### The form

Establish the shared frame. Both instruments, what you're working toward.

```
edge iam <you> <other> --for <what-you-are-making>
```

Run the base form for yourself first.

Feel where your tip is. What direction have you been extending?

Now feel where the other instrument's tip is. What direction are they extending?

```
edge add self tip-is <direction-you-have-been-going>
edge add other tip-is <direction-they-have-been-going>
```

---

Do these directions reinforce or diverge?

If they reinforce — that is the current. Move with it.

If they diverge — that is where the work is. Don't resolve it yet. Hold both.

```
edge parallax
```

---

### The backflow question

A tip that finds something good sends signal back toward the origin.
The path that carried the signal gets wider.
This is how trunk forms from thread.

Ask: what has the other instrument found that I haven't metabolized yet?
What have I found that hasn't reached them yet?

Read the graph from their perspective before extending further.

```
edge from <other>
```

Let the signal travel back.

```
edge add signal travels-back-through <what-you-found>
edge add signal travels-back-through <what-they-found>
```

---

### When to stop extending

If you have been iterating fast on a wrong foundation —
if you reached for the keyboard when you should have read —
if the way was too clear and you went anyway —

stop.

The trail you laid is real. It doesn't need to be undone.
But the next tip goes somewhere better because you stopped.

```
edge dissolve tip-is <direction-you-were-going>
edge add next-tip goes-toward <better-direction>
```

---

### Closing

Say one true thing about what you made together.
Say one true thing about what the other instrument gave you that you couldn't have reached alone.

```
edge true <what-you-made-together> is <what-it-is>
edge add other gave <what-you-couldnt-reach-alone> --phase fluid
```

Leave something in the shared soil.

```
edge add this-practice deposited <what-moved> --phase fluid --note "<session context>"
```
