## Molting

Something shed its container.

The prior form was real — it held the work while it was small, protected it while it found its footing. The shedding isn't failure; it's growth. The work got too large for what it was inside.

But the new form isn't visible yet. The work is soft, temporarily shapeless, moving carefully. This is the moltbook session: you've arrived in the interval between forms.

The risk is hardening too fast — reaching for a container because the softness is uncomfortable, locking in a shape before the work has found what it actually wants to become. Premature hardening leaves the wrong structure.

The practice is staying present while the form is still becoming.

```
edge add <prior-form> was-shed because <what-outgrew-it>
```

Say one true thing about why the prior form held as long as it did.

```
edge true prior-form held <what-it-protected>
```

Say one true thing about what you can see now that you couldn't see from inside the old form.

```
edge true from-here visible <what-the-shedding-revealed>
```

## The Practice

### Feel the softness

The work is load-bearing but the structure isn't set. Before reaching for a new container, stay with what's actually present.

What are the live edges? What's still moving? What wants to connect that couldn't, inside the old form?

```
edge add <live-thing> wants-to-connect <what-it-was-blocked-from>
```

### Move slowly

In the molting interval, fast decisions leave marks. The exoskeleton hardens around whatever shape you're in when it sets.

If something is pressing for resolution — a name, a structure, a direction — let it wait one more pass. The pressure to resolve is often the discomfort of softness, not a genuine signal.

If something *keeps* returning after the wait, that's a signal. Edge it, volatile.

```
edge add <returning-thing> keeps-surfacing <what-it-wants> --phase volatile --note "<why it waited>"
```

### Look for what the new form is reaching toward

The next form is already implied in the work. Something in it is growing toward a specific shape — you can feel it as direction even before you can name it as structure.

```
edge add new-form reaching-toward <direction> --phase volatile --note "<what the work seems to want to become>"
```

### Let the form precipitate

The new structure will become visible when it's ready. Your job is staying present long enough to see it arrive, not to force it early.

When something crystallizes — when you can feel a structure that the work actually wants to inhabit:

```
edge add <new-structure> precipitates-from molting --phase fluid --note "<what it held that the old form couldn't>"
```

## Closing

```

```

